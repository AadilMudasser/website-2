# Dr. Alia Qaisar Medical Consultation Website

Premium multi-page React frontend built as a static app.

## Open Locally

Open `index.html` in a browser.

The app uses React from the official UMD CDN and external image assets, so an internet connection is needed for the full visual preview.

## Pages

- Home
- About
- Weight Loss
- Women's Health
- Services
- Blog
- Book Consultation

## Key Interactions

- Sticky navigation
- Scroll reveal animations
- Horizontal services slider
- Interactive weight loss calculator
- Expandable services and modal details
- Responsive consultation form
